Αυτή είναι μια επιπρόσθετη ενότητα για την αλλαγή της περιοχής της SysNAND σας σε arm9loaderhax CFW. Αυτό επιτυγχάνεται εγκαθιστώντας την εικόνα της ctrtransfer 9.2.0 για την περιοχή στην οποία θέλετε να μεταφέρετε την συσκευή σας και έπειτα τροποποιώντας το αρχείο SecureInfo_A.
{: .notice--primary}

**Region changing is almost completely unnecessary since Luma3DS supports out of region games and individual [title region emulation](https://github.com/AuroraWright/Luma3DS/wiki/Options-and-usage).**
{: .notice--info}

Αυτή η διαδικασία θα αποσυνδέσει το NNID σας από το σύστημα, καθώς δεν θα είναι πλέον συμβατό με αυτό. Τα NNID είναι κλειδωμένα για την περιοχή της συσκευής στην οποία δημιουργήθηκαν και δεν μπορούν να μεταφερθούν μεταξύ περιοχών χωρίς [μια πολύ περίπλοκη και προηγμένη διαδικασία](https://gist.githubusercontent.com/yifanlu/e80db121d38aceb8cca0e03cefd5853b/raw/3c4dd89869156ca0f945a2791e699acfdb32b510/gistfile1.txt).
{: .notice--warning}

Μετά από αυτήν την διαδικασία, μόνο **τα New 3DS που δεν έχουν ξαναχρησιμοποιήσει το eShop πριν** και **οποιοδήποτε Old 3DS** θα μπορούν να αποκτήσουν πρόσβαση στο eShop μετά από την δημιουργία ενός καινούργιου NNID για την καινούργια περιοχή. New 3DS τα οποία είχαν χρησιμοποιήσει το eShop στην αρχική περιοχή της συσκευής δεν μπορούν να δημιουργήσουν καινούργιο NNID και να αποκτήσουν πρόσβαση στο eShop στην καινούργια περιοχή!
{: .notice--warning}

Σημειώστε ότι σποραδικά το eShop δεν θα λειτουργεί, χωρίς λόγο, σε συσκευές στις οποίες θα έπρεπε. Έχει να κάνει αποκλειστικά με του διακομιστές της Nintendo και δεν μπορώ να έχω καθόλου έλεγχο, όσον αφορά αυτό το θέμα.
{: .notice--warning}

Η αλλαγή της περιοχής με την χρήση της ctrtransfer (την οποία θα χρησιμοποιήσουμε) φαίνεται να προκαλεί την καταστροφή των ενημερώσεων επανεκκίνησης. Η αιτία είναι άγνωστη, αλλά προς το παρόν οι χρήστες Old 3DS δεν θα μπορούν να παίξουν παιχνίδια εκτεταμένης μνήμης (όπως το Monster Hunter, το Super Smash Bros, και το Pokémon Sun / Moon).
{: .notice--warning}

**ΠΡΕΠΕΙ να έχετε προεγκατεστημένα το arm9loaderhax και το Luma3DS για να προχωρήσετε.**
{: .notice--danger}

#### Οδηγίες

**Ορισμένες κονσόλες έχουν το αρχείο `SecureInfo_B` αντί για το `SecureInfo_A`· δεν υπάρχει πρόβλημα και θα πρέπει να χρησιμοποιήσετε το `SecureInfo_B` για τα σχετικά βήματα.**   
{: .notice--info}

##### Ενότητα I - Προετοιμασία

1. Αντιγράψτε το `GodMode9.bin` από το GodMode9 `.zip` στον φάκελο `/luma/payloads/` στην κάρτα SD και μετονομάστε το `GodMode9.bin` σε `up_GodMode9.bin`
2. Αντιγράψτε το `Decrypt9WIP.bin` στον φάκελο `/luma/payloads/` στην κάρτα SD και μετονομάστε το `Decrypt9WIP.bin` σε `x_Decrypt9WIP.bin`
3. Αντιγράψτε την εικόνα της ctrtransfer 9.2.0 `.bin` και `.bin.sha` από το ctrtransfer `.zip` στον φάκελο `/files9/` στην κάρτα SD
4. Επανατοποθετήστε την κάρτα SD μέσα στο 3DS

##### Ενότητα II - Αντίγραφα ασφαλείας των tickets και του config save

1. Πατήστε (Start) και έπειτα κρατήστε πατημένο το (X) για να εκκινήσετε στο Decrypt9 του arm9loaderhax     
3. Μεταβείτε στο "Ticket/Titlekey Options" και έπειτα επιλέξτε "Ticket Dump (SysNAND)"
4. Πατήστε (Select) για να αφαιρέσετε την κάρτα SD και έπειτα τοποθετήστε την στον υπολογιστή σας
5. Διαγράψτε όλα τα αρχεία `.tik` στον φάκελο `/files9/` τα οποία **δεν** αρχίζουν με "00-"
6. Αφήστε όλα τα αρχεία `.tik` που αρχίζουν με "00-" μέσα στον φάκελο
7. Επανατοποθετήστε την κάρτα SD στο 3DS και έπειτα πατήστε (B) για να επιστρέψετε στο Αρχικό Μενού
8. Μεταβείτε στο "SysNAND Options", έπειτα στο "System Save Dump" και μετά επιλέξτε "Dump configsave.bin"

##### Ενότητα III - ctrtransfer

1. Go to "SysNAND Options", then "CTRNAND Transfer", then "Auto CTRNAND Transfer"
2. Επιλέξτε την εικόνα ctrtransfer 9.2.0 όταν σας ζητηθεί πατώντας (A)
3. Εάν δεν έχετε ήδη αντίγραφα ασφαλείας της NAND **δημιουργήστε αντίγραφα ασφαλείας της SysNAND στο `NANDmin.bin` όταν σας ζητηθεί πατώντας (A)**
4. Επιτρέψτε στην διαδικασία της μεταφοράς να συνεχιστεί αυτόματα, η οποία μπορεί να διαρκέσει αρκετά λεπτά
5. Μόλις η μεταφορά ολοκληρωθεί, πατήστε (Select) για να εξάγετε την κάρτα SD
6. Put your SD card in your computer, then copy `NANDmin.bin` and `NANDmin.bin.sha` from the `/files9/` folder on your SD card to a safe location
  + Δημιουργήστε αντίγραφα ασφαλείας σε πολλαπλές τοποθεσίες
  + This backup will save you from a brick if anything goes wrong in the future
7. Delete the 9.2.0 ctrtransfer image `.bin` and `.bin.sha` from the `/files9/` folder on your SD card after copying it
8. Επανατοποθετήστε την κάρτα SD μέσα στο 3DS

##### Section IV - Editing SecureInfo

1. Press (Start) then hold (Up) to reboot into arm9loaderhax GodMode9      
2. Navigate to `SYSNAND CTRNAND` -> `rw` -> `sys`
3. Press (Y) on `SecureInfo_A` to copy it
4. Press (Y) to paste a copy of `SecureInfo_A`
5. Select "Copy path(s)"
6. Press (A) to unlock SysNAND writing, then input the key combo given
7. Select "Choose new name"
8. Use the D-Pad to rename the file to `SecureInfo_C`, pressing (A) when you're done (overwrite any existing `SecureInfo_C`)
9. Scroll down to the `SecureInfo_C` you just pasted
10. Press (A) on `SecureInfo_C` then select "Show in Hexeditor"
11. Press (A) to continue
12. Press (A) to enter edit mode
13. Go to the beginning of line 00000100 and press (A) on it
14. Hold (A) while pressing (Up) or (Down) on the D-Pad to edit the number
15. Change the first number pair on line 00000100 to the following pair that corresponds to *the region you want to change to*:
    - "00" : JPN
    - "01" : USA
    - "02" : EUR
16. Press (B) to exit edit mode, then save changes, then press (B) again to return to the directory
17. You should see both a `SecureInfo_A` and `SecureInfo_C` if everything was done properly. (`SecureInfo_C` is used in place of `SecureInfo_A` when using Luma3DS if it detects the file on boot)
18. Press (Start) to reboot

##### Section V - Reinstalling Tickets

1. Open FBI
2. Select "SD"
3. Select "files9"
4. Select "\<current directory>"
5. Select "Install and delete all tickets"
6. Wait. The system may appear to freeze, just give it time.
7. Press (A) to confirm
8. Press (B) to decline installing tickets from CDN.
9. Exit with the home button

##### Section VI - Region settings

1. Open the System Settings
2. Go to "Other Settings", "Profile", then "Region Settings"
3. Select a country from the region you switched to
4. If prompted, you do not need to set a state
5. Update your region changed CFW SysNAND to the latest version

##### Section VII - Restore config save

1. Πατήστε (Start) και έπειτα κρατήστε πατημένο το (X) για να εκκινήσετε στο Decrypt9 του arm9loaderhax   
2. Go to "SysNAND Options", then "System Save Inject", then select "Inject configsave.bin"

Old 3DSs / 2DSs may need to format (using TinyFormat or the System Settings) to play extended memory mode games (such as Monster Hunter, Super Smash Bros, or Pokemon Sun/Moon).
{: .notice--info}

Formatting a device following a ctrtransfer *without restoring the config save* can cause screen brightness / flickering issues. See [this](https://github.com/Plailect/Guide/issues/794).
{: .notice--info}
